Luminary SaaS Dashboard

A comprehensive analytics dashboard redesign that increased DAUs by 41% and reduced support tickets by 34%.

Client
Luminary Inc.
Role
Lead Designer & Developer
Timeline
8 Weeks
Year
2024

The Challenge

Luminary's existing dashboard was built by engineers without design input — functional, but dense and confusing, with a 68% abandonment rate during onboarding. Our goal was to redesign the entire experience from the ground up, turning complexity into clarity without sacrificing power.

The Process

We started with 14 user interviews across three customer segments — power users, casual users, and new signups. Each group had fundamentally different needs and mental models, which shaped every information architecture decision.

The IA was restructured around "jobs to be done" rather than data categories. We created a flexible layout system using CSS Grid that lets users customise their dashboard without any code.

The Solution

The redesigned dashboard delivers the right information at the right time. A progressive disclosure model keeps the interface clean for beginners while giving power users everything they need with a single click.

Performance was critical — we reduced initial load time by 62% through code splitting, lazy-loaded charts, and a skeleton loading strategy that gives users instant perceived performance.

Results

Six weeks after launch: 41% increase in daily active users, 34% reduction in support tickets, and 28% increase in trial-to-paid conversion. The client called it "the single best investment we've made in 2024."
